# Galeri klasörleri

- **`pencil/`** — Kalem çizimlerini buraya koyun. Popup’ta “Pencil” seçildiğinde listelenir.
- **`brush/`** — Fırça çizimlerini buraya koyun. “Brush” seçildiğinde listelenir.

Desteklenen formatlar: PNG, JPG, JPEG, WebP, GIF.

Dosyalar alfabetik sırayla gösterilir (ör. `01.png`, `02.png` …).
